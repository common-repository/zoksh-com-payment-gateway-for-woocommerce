= Plugin installation =

1. Upload the zoksh-payment-gateway-for-woocommerce directory to the /wp-content/plugins/directory.
2. Activate the plugin through the “Plugins” menu in WordPress.

After the plugin is activated, Zoksh will appear in the WooCommerce>Payments section.

= Plugin configuration =

After you have installed the plugin, the configuration steps are:

1. On the WooCommerce “Settings” page, go to the “Payment Gateways” tab, then click "Zoksh"
2. Check “Enable Zoksh" and enter your Zoksh Api Key & Secret.
3. Click “Save changes” and the gateway will be active.
